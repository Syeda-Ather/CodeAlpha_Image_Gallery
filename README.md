# Interactive Image Gallery

Welcome to the **Interactive Image Gallery** project! This repository contains the source code for Task 1 of the **CodeAlpha Frontend Development Internship**.

## Overview
This is a fully responsive and interactive image gallery featuring dynamic category filtering and a sleek lightbox viewer. It provides a smooth user experience across all devices.

## Features
- **Responsive Grid Layout:** Adapts perfectly to mobile, tablet, and desktop screens.
- **Dynamic Filtering:** Sort images by category (Nature, City, Abstract, Travel).
- **Lightbox View:** Full-screen viewing with navigation controls.
- **Keyboard Support:** Navigate using arrow keys; close using the Escape key.
- **Smooth Animations:** Clean hover effects and transition animations.

## Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Flexbox/Grid layouts, custom styling, and animations.
- **JavaScript:** DOM manipulation for image filtering and lightbox logic.
